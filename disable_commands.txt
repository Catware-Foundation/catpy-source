Отключённые команды бота

чтозааниме: Поиск аниме по приложенному изображению.

импакт: Создание мемов с текстом сверху и снизу. Использование: верхний текст;нижний текст

скрин: Скриншотит сайт при помощи roughs.ru

раскладка: Меняет раскладку текста

дшип: Шипперство рандомных парочек в беседе - несколько штук за раз (по умолчанию 3, максимум 10)

ошибка: Генератор ошибок Windows XP

мотиватор: Как демотиватор, только мотиватор

браузер: Получает текст веб-страницы

кнопки: Добавить кнопку в диалог (Используйте флаг -delete для удаления кнопок.)

пикчи: Поиск картинок (отдельное спасибо vk.com/opensnuse)

дуракшифр: Такой себе шифр (команды дешифр/шифр)

гс: Выводит текст голосового сообщения.
