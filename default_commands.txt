Команды бота

транслит: Транслитерация текста на русском языке

погода: Поиск прогноза погоды для населенного пункта.

подключить-пакет: Подключить пакет потенциально оскорбительных команд.

кумыс: Напаивает фотографию кумысом

лобстер: Наложение шрифта Lobster на изображение.

курс: Выводит курс валют по данным ЦБ РФ

dehex: Декодировщик текста из HEX

ник: Позволяет просмотреть или изменить свой ник

рулетка: Пытается застрелить всех участников беседы, но убивает только одного. Есть более жестокий аналог: /чат рулетка

зальго: Превращает текст в зальго

олбан: Олбанизатор, коверкойущий всйо в олбанскей

араб: разговаривать с арабом

город: Сохраняет/показывает населённый пункт по умолчанию для некоторых команд

цитата: Генерирует цитату из сообщения

капча: undefined

вкид: Стилизирует ваш никнейм под доменное имя ВКонтакте

репорт: Отправить сообщение администрации CatPy Stable. Также можно прикладывать скриншот к репорту

расстояние: Измеряет расстояние между двумя локациями

кси: Команда который превращать ваш русский в ломать русский

приветадминам: Передает привет админам котопая от юзера

перефраз: Перефразирование текста с помощью Google Переводчика

фотовтекст: Распознаёт текст на фото и выводит его

qr: Сгенерировать qr код

цп: Информация о процессоре - замеры на GeekBench

хрю: Переводит текст в украинский

биткоин: Текущий курс биткоина

шип: Шипперство рандомных парочек в беседе

дешифр: Дешифровка Котошифра

бред: Генерация рандомного бреда на основе различных текстов

поиск: Поисковик

донат: Задонатить в Catware

тест: Тестовая команда

ошибка: Создание ошибки в стиле Windows XP. Использование: ошибка заглавие;текст;кнопка1;кнопка2;кнопка3

скажи: Попросить сказать что либо

время: Показывает время

команды: Список команд бота.
Использование: команды (флаг)
Флаги:
-откл, -доп, -лист

чат: Функционал для админов бесед

новости: Свежие новости

перевод: Перевод текста с помощью Google Translate
Использование команды: перевод -<рус/анг/нем> <текст>

текст: Поиск текста песни в базе данных Genius.

инфо: Информация о боте и разработчиках

демотиватор: Создание демотиваторов. Используйте ; для разделения текста

шифр: Шифрование в Котошифр

скачатьаудио: Скачать трек или голосовое сообщение из ВКонтакте

enshort: Сократитель английских слов и названий. К примеру, catware -> ctwr

символ: Считает встречаемость каждого символа в тексте

имт: Рассчитывает ваш индекс массы тела. Использование: /имт [рост в см] [вес в кг]

пб: Меняет первые буквы в предложении (пример: пачка чипсов -> чачка пипсов). Действует строго рандому

помощь: Помощь с catpy

орфография: Проверяет текст на наличие орфографических ошибок на базе Яндекс.Спеллера

выбери: Выбирает один элемент из нескольких перечисленных через "или" элементов

стат: Некоторая интересная статистика.

вычислить: Вытягивает всю инфу про айпи из WHOIS.
Использование: /вычислить имяхоста

q: Добавляет сообщение в цитатник q.ctw.re

гео: Команда получения географических координат по точному адресу места. 

кек: Разговаривать с идиотом

hex: Кодировщик текста в HEX

вики: Поиск в Википедии
Используйте флаг -кратко для сокращения информации в размер 4 предложений.

котейка: Пикча котейки с thiscatdoesnotexist.com

место: Поиск информации по геолокационной метке

нг: Считает, когда наступит новый год для города, указанного в команде. Если город не был указан в команде, рассчитывает для города в профиле.

средняяоценка: Вычисляет среднюю оценку для последовательности оценок вида 545523225

ютуб: Ищет каналы и видео на видеохостинге YouTube. Укажите флаг "-видео" в конце для поиска видео

ковид: Выводит последнюю доступную статистику коронавируса по России

цвета: Реверсировать цвета на картинке

перелом: Сломать фотографию

сэм: Записывает голосовое сообщение голосом Microsoft Sam

ёда: Перемешка слов

праздники: Отправляет картинку с текущими праздниками с calend.ru

шанс: Прикидывает вероятность того или иного события

gr: Генерирует код ошибки по тексту.

прф: Продолжение строчки на основе бредогенератора

бравл: Статистика игрока Brawl Stars по тэгу
